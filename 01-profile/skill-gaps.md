## Bridging Plan

### SQL
- **Resource:** Mode Analytics SQL Tutorial (free) / SQLZoo / LeetCode Easy problems
- **Milestone:** Complete 30 SQL problems → add to resume/SOP
- **Time:** 4–6 weeks

### Python
- **Resource:** Python for Everybody (Coursera, University of Michigan) — free audit
- **Milestone:** Complete modules 1–4, build 1 small data analysis project
- **Time:** 6–8 weeks

### Statistics
- **Resource:** Khan Academy Statistics / Think Stats (free book)
- **Milestone:** Comfortable with distributions, hypothesis testing, regression concepts
- **Time:** 4 weeks alongside SQL

### Marketing Analytics / Tableau
- **Resource:** Tableau Public (free), Google Analytics Certification (free)
- **Milestone:** Google Analytics cert + 1 Tableau dashboard project
- **Time:** 3–4 weeks
