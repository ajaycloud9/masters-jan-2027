# Financial Proof Documents

Place all financial proof documents in this folder when the time comes.

## What to Prepare

| Document | When Needed | Status |
|---|---|---|
| Bank statements (6 months) | Visa application | ⬜ |
| Fixed deposit certificates | Visa application | ⬜ |
| Sponsor's income proof (salary slips / ITR) | Visa application | ⬜ |
| Scholarship letter from university | Visa application | ⬜ |
| Education loan sanction letter (if applicable) | Visa application | ⬜ |

## Funding Strategy

Decide which funding route to use:
- [ ] Family funds (bank account of parent/guardian)
- [ ] Education loan (from Indian bank)
- [ ] Combination of both
- [ ] Scholarship from university

> Note: Having a sanction letter from a recognized Indian bank (SBI, HDFC, ICICI) for an education loan is strong evidence of financial commitment and also demonstrates ties to India (loan repayment obligation).
